# SignLang-Vision
 An AI-powered real-time Indian Sign Language (ISL) recognition web app built with Python, MediaPipe, and Machine Learning.  
 ## Features
 - 🎥 Real-time hand gesture detection via webcam
 - 🔤 Recognizes ISL alphabets (A-Z)
 - 📚 Learn ISL gestures reference chart
 - 💬 User feedback form
